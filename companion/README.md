# companion — the twin companion: genetics, breeding, and bones

An installable no-backend PWA (`manifest.webmanifest` + `sw.js`) where the twin
lives as a creature: `genetics.mjs` derives heritable traits from content hashes,
`twin.mjs` speaks for the bones, and the `agents/` directory carries the Python
agent bodies (`brain.py`, `agent_runtime.py`) that a local brainstem can run.
