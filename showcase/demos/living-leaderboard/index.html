    Why it's cool

    A poster is dead ink. But this one keeps moving: every scan makes the visitor's browser verify code bytes locally, fetch a current public ranking, and draw a fresh top-10 leaderboard. The artifact is physical, the board is alive, and nobody runs a backend in the middle. The same printed QR keeps updating as the world changes.

    Run it yourself

    1. Open /track, keep the pinned extract cell, and set src to https://raw.githubusercontent.com/EvanLi/Github-Ranking/master/Top100/JavaScript.md.
    2. Use path as blank so the full markdown table is returned, then parse rows into top entries in-page.
    3. Encode the spec into a QR and print it. Every scan re-fetches and re-ranks in the scanner's browser.

    Local: python3 -m http.server in the repo root, then open localhost:8000/showcase/demos/living-leaderboard/ (crypto.subtle needs a secure context — localhost or https, not file://).
