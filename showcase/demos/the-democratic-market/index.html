    Why it's cool

    The Permanent Market already gives an unkillable, notarized, append-only marketplace. The Trustless Poll already gives ECDSA-signed ballots that reach append-only consensus with no coordinator. This weld keeps both intact and joins them at governance: listing/moderation approval is now a verifiable poll ledger, not an admin toggle.

    So every transaction has two tamper-proof artifacts: a governance verdict (who signed what, and in what append-only order) and the resulting sealed/custody-stamped/seller-signed market order. Replay the chain and you can verify both economic action and democratic rule-setting.

    Run it yourself

    1. Press Resolve mirrors + load live market. The page should prove the same pinned cell survives across mirrors and then load live sold-out labels from GitHub Issues.
    2. Select at least one available item and press Run trustless governance poll. The page mints ECDSA ballots, runs append-only fast-forward consensus, and verifies the tally from the accepted ledger.
    3. Only after a YES majority ratifies the cart, press Finalize governed order + append to permanent ledger.
    4. Watch the order become an RSL1 envelope, a custody link, a seller-signed receipt, and finally a permanent ledger entry that embeds the governance verdict.
    5. Repeat with another order: the ledger should append, not replace, and the verifier should replay every prior entry before accepting the new head.
    6. Press Tamper latest ledger entry to mutate stored content. Verification should halt on that exact entry.

    Local: python3 -m http.server in the repo root, then open localhost:8000/showcase/demos/the-democratic-market/ (crypto.subtle needs localhost or https, not file://).
