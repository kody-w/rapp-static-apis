The Night You Were Born

Live — run it right now

Give it a date and coordinates. No API calls. The stars are a small inlined catalog of real RA/dec points, rotated by local sidereal time; the moon phase comes from the date.

animation: sidereal drift (quietly turning)
— press the button —

Why it's cool

This is a memory engine with no memory. Date + place are enough to reconstruct a sky forever: same stars, same moon phase, same geometry over the horizon. It feels personal, but the source of truth is just celestial mechanics.

Run it yourself

Local: python3 -m http.server in the repo root, then open localhost:8000/showcase/demos/the-night-you-were-born/.
