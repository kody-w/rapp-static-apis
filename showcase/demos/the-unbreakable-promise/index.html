The Unbreakable Promise

Live — run it right now

Type a promise. Your browser fetches a pinned cell from /track and verifies its hash before running, then generates a fresh ECDSA P-256 keypair, signs your exact text with crypto.subtle, and re-verifies that signature immediately. No server stores or decides anything.

    Why it's cool

    This is a vow with cryptographic teeth. The private key never leaves your browser, yet anyone can later check the shared payload and prove whether the exact message was signed by the exact public key. You cannot "take it back" by editing words after the fact: one character changes the bytes, and verification fails. It's receipts for promises.

    Run it yourself

    1. Open this page, type a commitment, and press Seal this promise.
    2. Copy the generated JSON receipt (message + JWK public key + signature).
    3. Share it anywhere. Anyone can paste it into their own browser console and verify with crypto.subtle.verify.

    Local: python3 -m http.server in repo root, then open localhost:8000/showcase/demos/the-unbreakable-promise/ (WebCrypto requires secure context: localhost or https, not file://).
