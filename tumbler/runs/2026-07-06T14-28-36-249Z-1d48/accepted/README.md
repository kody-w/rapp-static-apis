# a small corpus, kept close

three channels, three posts — small enough to hold in one hand: welds, seals,
holograms. every record carries an `id`, and the id is sacred.

the tumbler tightens the prose around each id but never touches the name. it may fix
a typo, close a double space, settle an old inconsistency — it may never add, drop,
or rename a single record.

loud loops chase a score; this one presses a flower.
