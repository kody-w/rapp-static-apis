# Demo corpus

This small corpus powers the tumbler **data-mode** demo. It holds **two** channels
(welds, seals, holograms) and three posts. Every record carries an `id` that is
sacred: the tumbler must preserve every id byte-for-byte while it tightens the prose
around it. The polish pass is allowed to fix spelling, spacing, and inconsistencies —
but it may never add, drop, or rename a single record.
